# 📊 HR Analytics Dashboard (Excel Project)

## 🔍 Overview

This project is an interactive **HR Analytics Dashboard** created using Microsoft Excel. It provides insights into employee data such as income, performance, job roles, and attrition. The dashboard is designed with a professional **Black & Gold theme** for better visualization and presentation.
<img width="1826" height="734" alt="HR DATA DASHBOARD" src="https://github.com/user-attachments/assets/3046ed39-e6dc-4cc2-bbcc-7d14c831cb83" />


## 🎯 Objectives

* Analyze employee salary distribution
* Understand performance ratings
* Track employee count across job roles
* Study attrition trends
* Provide interactive and dynamic insights
  
## 📁 Dataset

The dataset includes the following attributes:

* Age
* Job Role
* Department
* Monthly Income
* Daily Rate
* Performance Rating
* Attrition (Yes/No)
* Gender
* Business Travel

## 🛠️ Tools & Technologies

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Cleaning
  
## 📌 Dashboard Features

### 🔸 KPI Cards

* Monthly Income
* Monthly Rate
* Standard Hours
* Daily Rate

### 🔸 Filters (Slicers)

* Age
* Job Role
* Attrition

### 🔸 Visualizations

* Performance Rating (Pie Chart)
* Monthly Income by Department (Bar Chart)
* Daily Rate by Job Role (Column Chart)
* Employee Count (Line Chart)
* Employee Distribution by Travel (Trend Chart)
* Job-wise Employee Count by Gender (Bar Chart)

## 📊 Key Insights

* R&D department has the highest income contribution
* Laboratory Technicians have the highest employee count
* Sales Executives show strong workforce presence
* Performance ratings are evenly distributed
* Employee trends vary based on travel frequency
